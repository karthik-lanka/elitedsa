30. Shortest Subarray with Target Sum or More

You're given: An array nums of positive integers A positive integer target Your task is to find the length of the smallest contiguous subarray such that the sum is greater than or equal to target. If no such subarray exists, return 0. Examples: Example 1: plaintext Copy Edit Input: target = 7, nums = [2,3,1,2,4,3] Output: 2 Explanation: The subarray [4,3] adds up to 7, which meets the requirement. Its length is 2, which is the shortest possible. Example 2: plaintext Copy Edit Input: target = 4, nums = [1,4,4] Output: 1 Explanation: The single-element subarray [4] is valid and minimal. Example 3: plaintext Copy Edit Input: target = 11, nums = [1,1,1,1,1,1,1,1] Output: 0 Explanation: No subarray sums to 11 or more.

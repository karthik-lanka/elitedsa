23. Locate First Occurrence of a Substring

You're given two strings: needle and haystack. Your task is to find and return the index of the first occurrence of needle within haystack. If needle does not appear in haystack, return -1.

Example 1: Input: haystack = "sadbutsad", needle = "sad" Output: 0 Explanation: The substring "sad" appears at indices 0 and 6. Since we want the first occurrence, we return 0.

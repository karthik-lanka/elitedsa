3. Remove Duplicates from a Sorted List

You are given a sorted integer array nums with non-decreasing order, and you need to remove duplicates in place, so that each unique element appears only once. The relative order of elements should remain unchanged. Then, return the number of unique elements in the array.

Example 1: Input: nums = [1, 1, 2] Output: k = 2, nums = [1, 2, _] Explanation: After removing duplicates, the unique elements are 1 and 2, so the function returns k = 2. The underscores represent elements that no longer matter.

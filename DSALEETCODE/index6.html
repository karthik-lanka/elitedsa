6. Shift Elements of an Array in Circular Fashion

Given an integer array nums, rotate the array to the right by k steps, where k is non-negative. The rotation should be done in-place, and no extra array should be allocated.

Input: nums = [1, 2, 3, 4, 5, 6, 7], k = 3 Output: [5, 6, 7, 1, 2, 3, 4] Explanation: Rotate 1 step to the right: [7, 1, 2, 3, 4, 5, 6] Rotate 2 steps to the right: [6, 7, 1, 2, 3, 4, 5] Rotate 3 steps to the right: [5, 6, 7, 1, 2, 3, 4]

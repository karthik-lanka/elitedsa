12. Design a Constant-Time Set with Random Access

To implement the RandomizedSet class with the required operations (insert, remove, and getRandom) in average O(1) time complexity, we can use a combination of a hashmap (for constant-time lookups) and an array (to maintain random access for getRandom).

randomizedSet = RandomizedSet() print(randomizedSet.insert(1)) # True print(randomizedSet.remove(2)) # False print(randomizedSet.insert(2)) # True print(randomizedSet.getRandom()) # Random output: 1 or 2 print(randomizedSet.remove(1)) # True print(randomizedSet.insert(2)) # False print(randomizedSet.getRandom()) # 2

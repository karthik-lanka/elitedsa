10. Minimum Jumps to Reach the End

You are given a 0-indexed integer array nums of length n. Initially, you start at nums[0]. Each element nums[i] indicates the maximum number of steps you can jump forward from index i. Specifically, if you are at index i, you can jump to any index i + j where: 0 <= j <= nums[i] i + j < n Your task is to determine the minimum number of jumps needed to reach nums[n - 1]. The test cases are constructed in such a way that reaching the last index is always possible.

Example 1: Input: nums = [2, 3, 1, 1, 4] Output: 2 Explanation: To reach the last index, you can jump: From index 0 to index 1 (1 step). Then from index 1 to index 4 (3 steps), reaching the last index.

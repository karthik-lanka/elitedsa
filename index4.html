4. Remove Extra Duplicates from a Sorted List (Allowing At Most Two)

You are given a sorted integer array nums, and your task is to remove some duplicates in-place so that each unique element appears at most twice. The relative order of the elements must be preserved. Since the array cannot be resized in some languages, your result should be placed in the first part of the array. Specifically, after removing duplicates, the first k elements of the array should hold the final result. It doesn't matter what you leave beyond the first k elements. Return k, the number of elements in the array after removing duplicates.

Example 1: Input: nums = [1, 1, 1, 2, 2, 3] Output: k = 5, nums = [1, 1, 2, 2, 3, _] Explanation: The function should return k = 5, with the first five elements of nums being 1, 1, 2, 2, and 3, respectively. The remaining elements (denoted by underscores) don't matter.

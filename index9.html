9. Determine Reachability in a Jump Array

JYou are given an integer array nums where each element represents your maximum jump length at that position. You start at the first index (nums[0]), and you need to determine whether it is possible to reach the last index.

Example 1: Input: nums = [2,3,1,1,4] Start at index 0. The maximum jump length is 2. So, farthest = max(0, 0 + 2) = 2. At index 1, farthest = max(2, 1 + 3) = 4. At index 2, farthest stays 4. At index 3, farthest stays 4. At index 4, farthest = 4, which is the last index.

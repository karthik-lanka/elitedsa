2. Eliminate Specific Value from Array

You are given an array of integers nums and a number val. Your goal is to remove all occurrences of val from the array and return the number of remaining elements that are not equal to val. After removing val, you should modify the array so that the first k elements (where k is the number of remaining elements) contain the values that are not val. The rest of the array doesn't matter.
